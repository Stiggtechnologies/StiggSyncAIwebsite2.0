# AIM META ADS - QUICK LAUNCH GUIDE

**After pixel installation is complete, follow these steps:**

---

## ✅ STEP 1: VERIFY PIXEL (2 Minutes)

1. Install **Meta Pixel Helper** Chrome extension:
   https://chrome.google.com/webstore/detail/meta-pixel-helper/fdgfkebogiimcoedlicjlajpkdmockpc

2. Visit: https://albertainjurymanagement.ca

3. Click Pixel Helper icon → should show:
   - ✅ "PageView event found"
   - ✅ "Pixel ID: 1492939798724540"

---

## ✅ STEP 2: UPLOAD CREATIVE ASSETS (5 Minutes)

**Need at minimum:**
- 3-5 images (clinic, treatment room, staff)
- OR use stock photos temporarily

**Where to upload:**
1. Go to: https://business.facebook.com/adsmanager/manage
2. Click "Create" → "Ad"
3. Upload images when creating ads

---

## ✅ STEP 3: CREATE CAMPAIGNS (10 Minutes)

### Campaign 1: Lead Generation

1. **Ads Manager** → **Create** → **Leads**
2. **Campaign Name:** AIM - Lead Gen - Core Services
3. **Budget:** Campaign budget OFF (use ad set budgets)
4. **Continue**

### Ad Set 1: Physiotherapy

5. **Ad Set Name:** Physio - Edmonton North
6. **Conversion Location:** Website
7. **Pixel:** Select "1492939798724540"
8. **Budget:** $12/day
9. **Location:** Edmonton, AB + 10km radius
10. **Age:** 25-65
11. **Interests:** Add "Health & wellness", "Physical fitness"
12. **Placements:** Automatic (or Facebook + Instagram Feed/Stories)
13. **Continue**

### Create Ads

14. **Identity:** Select AIM Facebook Page (or create one)
15. **Format:** Single Image or Carousel
16. **Upload images**
17. **Primary Text:** Copy from `AIM-META-CAMPAIGNS-READY-TO-LAUNCH.md`
18. **Headline:** Copy from same doc
19. **CTA Button:** "Call Now" or "Book Now"
20. **Website URL:** https://albertainjurymanagement.ca
21. **Publish**

**Repeat for MVA and WCB ad sets** (10 min each)

---

## ✅ STEP 4: SET BUDGET LIMITS (2 Minutes)

1. **Ads Manager** → **Settings** → **Account Spending Limit**
2. **Set:** $100/day maximum
3. **Save**

---

## ✅ STEP 5: ACTIVATE & MONITOR (Ongoing)

1. **Activate Campaign 1** (Lead Gen - Core)
2. **Wait 24-48 hours** for initial data
3. **Check daily:**
   - Spend vs budget
   - Leads generated
   - Cost per lead
4. **If good:** Activate Campaign 2 (Retargeting)
