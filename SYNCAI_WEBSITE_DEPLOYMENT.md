## 🌐 CUSTOM DOMAIN SETUP (syncai.ca)

To configure the custom domain:

### Step 1: Add Domain in Vercel
1. Go to https://vercel.com/dashboard
2. Select project: **syncai-website**
3. Go to **Settings** → **Domains**
4. Add domain: `syncai.ca`
5. Vercel will provide DNS configuration

### Step 2: Configure DNS at Registrar

**Option A - Recommended (CNAME):**
```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
TTL: 3600
```

**Option B (A Record for apex):**
```
Type: A
Name: @ (or syncai.ca)
Value: 76.76.21.21
TTL: 3600
```

### Step 3: SSL Certificate
Vercel automatically provisions SSL certificates once DNS propagates (usually 5-30 minutes).
