# SyncAI Sales Execution Plan (Outbound) — Food & Beverage Manufacturing (ON/QC)

**Objective:** Acquire first **10–15 customers** (Y1 target 15) in **Ontario + Quebec** Food & Beverage manufacturing.

**Deal model assumptions (from ICP research):**
- Sales cycle: **45–75 days**
- Target win rate (qualified opps): **30–35%**
- Core plan sold: **Business tier ~ $2,499/mo ($29,988/yr)**
- Motion: **Founder-led outbound → ROI assessment → demo → pilot (60 days) → annual**

## What’s in this folder

1) **Prospecting** (`prospecting/`)
- Prospect DB schema (CSV)
- Scoring model
- A repeatable workflow to build **500+ qualified accounts + contacts**

2) **Outbound sequences** (`sequences/`)
- LinkedIn 7-touch
- Email 5-touch
- Call scripts + voicemails
- Recommended cadence and daily activity targets

3) **Sales collateral** (`collateral/`)
- One-pager
- Demo script (30 min) + discovery
- Pilot offer + success criteria
- ROI assessment talk track (uses ROI_CALCULATOR)
- Case study template + mutual action plan

4) **Pipeline infrastructure** (`crm/`)
- CRM setup, stages, required fields
- Dashboards, reports
- Opportunity playbook

5) **Execution plan (90 days)** (`execution/`)
- Week-by-week plan
- Daily targets
- Conversion tracking + leading indicators

---

## Quick-start (do this in order)
1. **Build the first 100 accounts** using `prospecting/BUILD_500_PROSPECTS.md` (ON/QC, 100–1,000 employees).
2. Score + segment accounts using `prospecting/SCORING_MODEL.md`.
3. Run sequences in `sequences/` to create **8–12 meetings/week** target.
4. Use `collateral/DEMO_SCRIPT.md` + `collateral/PILOT_OFFER.md` to convert demos → pilots.
5. Track all activity + conversion metrics per `execution/90_DAY_PLAN.md`.
