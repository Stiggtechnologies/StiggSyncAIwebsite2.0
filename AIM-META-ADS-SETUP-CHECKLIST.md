## STEP 7: SET AUTOMATED RULES (3 minutes)

1. In Ads Manager, click **"Tools"** (top menu)
2. Click **"Automated Rules"**
3. Click **"Create Rule"**

### Rule 1: Pause High CPL
- **Rule name:** Pause if CPL >$50
- **Apply to:** All active ad sets
- **Conditions:**
  - Cost per result > $50
  - AND Spend > $30
- **Action:** Pause ad set
- **Notification:** Send email notification
- **Schedule:** Continuous

### Rule 2: Scale Winners
- **Rule name:** Scale if CPL <$25
- **Apply to:** All active ad sets
- **Conditions:**
  - Cost per result < $25
  - AND Spend > $50
- **Action:** Increase daily budget by 20% (max $35)
- **Notification:** Send email notification
- **Schedule:** Daily at 8:00 AM

Click **"Create"** for each rule

**"Sunset Manor" still showing?**

**Images won't upload?**
- Check file size (should be <30MB)
- Try different browser
- Upload one at a time

**Campaign won't activate?**
- Check payment method in Business Manager
- Verify billing threshold not exceeded
- Check for policy violations

**No leads coming in?**
- Check form is connected properly
- Verify email notifications set up
- Test lead submission manually
