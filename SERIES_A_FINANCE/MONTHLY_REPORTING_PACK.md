# Monthly Finance Reporting Pack (Template)

## 1. Executive Summary (1 page)
- Highlights (wins, risks, asks)
- ARR, net new ARR, customers
- Burn, runway

## 2. Financials
- P&L actual vs budget (month + YTD)
- Cash reconciliation (beginning cash, inflows, outflows, ending cash)
- Expense breakdown by function

## 3. SaaS metrics
- ARR bridge (start ARR + new + expansion - churn = end ARR)
- GRR/NRR and churn
- CAC + payback (if enough data)

## 4. GTM
- Pipeline by stage + expected close date
- New leads/MQL/SQL
- Win/loss notes and learnings

## 5. Operating plan
- Headcount actual vs plan
- Hiring pipeline for key roles
- Roadmap milestone status

## 6. Appendix
- Bank statement snapshot
- Top vendors and spend
- Contracted ARR schedule
