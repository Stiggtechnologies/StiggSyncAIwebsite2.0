# SyncAI — Series A Data Room Index (Checklist)

## 0. Read-me
- Company overview + quick links
- Contact list

## 1. Corporate / Legal
- Certificate of incorporation + bylaws
- Shareholder agreements
- Board minutes + consents
- IP assignment agreements (all founders/employees/contractors)
- Material contracts (customer, vendor, partner)

## 2. Cap Table / Financing
- Current cap table (by class)
- Option plan docs + grants
- Prior SAFEs/notes + conversion terms
- Proposed Series A terms (draft)

## 3. Financials
- Historical P&L / cash (even if minimal)
- Budget (next 12/24 months)
- 5-year model (this package)
- Bank statements (last 6–12 months)

## 4. Go-To-Market
- ICP definition + segmentation
- Pipeline report (CRM export)
- Sales deck + pricing
- Customer ROI calculator + case study template

## 5. Product / Technology
- Product demo video + walkthrough
- Architecture diagram
- Security overview
- Roadmap (12–18 months)
- Model performance metrics (precision/recall, false positives)

## 6. Team
- Org chart + hiring plan
- Key hires needed + role scorecards
- Advisor list

## 7. Market
- Market sizing
- Competitive analysis

## 8. Compliance / Risk
- Data handling policies
- Incident response plan
- SOC2 readiness plan (if applicable)
- Insurance (general liability, cyber)
