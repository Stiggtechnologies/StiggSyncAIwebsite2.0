# Operational Finance — Metrics Dashboard Spec (Series A-ready)

## Cadence
- Weekly: pipeline + bookings + cash
- Monthly: ARR movement, cohort retention, burn/runway, unit economics
- Quarterly: plan vs actual, headcount, roadmap vs milestones

## Core KPIs (board-ready)
### Growth
- ARR / MRR
- New ARR, expansion ARR, churn ARR (ARR bridge)
- Customers: new, churned, total
- ARPA / ACV

### Efficiency
- CAC (blended + by channel)
- CAC payback
- LTV and LTV:CAC
- Magic Number
- Sales cycle length

### Retention
- Gross Revenue Retention (GRR)
- Net Revenue Retention (NRR)
- Logo churn
- Cohort retention curves

### Profitability / Cash
- Gross margin
- Burn (net cash out)
- Runway (months)
- Burn multiple ($ burned / net new ARR)

### Sales execution
- Pipeline coverage (next 90 days)
- Win rate, stage conversion rates
- AE quota attainment

## Recommended dashboard layout
1. Executive summary tiles (ARR, NRR, burn, runway)
2. ARR bridge waterfall
3. Funnel + pipeline by stage
4. Cohort retention
5. Unit economics table
6. Headcount plan vs actual

## Data sources
- Billing (Stripe/Chargebee)
- CRM (HubSpot/Salesforce)
- Product analytics (events, connected assets)
- Accounting (QuickBooks/Xero)
