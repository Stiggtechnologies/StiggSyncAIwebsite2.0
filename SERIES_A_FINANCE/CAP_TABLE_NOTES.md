# Cap Table & Valuation — Framework (Series A)

## Inputs to decide
- **Pre-money valuation** (e.g., $30M–$45M)
- **Raise size** (e.g., $6M–$10M)
- **ESOP / option pool** size (commonly 10–15% post-money at Series A)
- Treatment of existing SAFEs/notes (convert, MFN, etc.)

## Simple dilution math (rule of thumb)
- **Investor ownership (post)** ≈ Raise / (Pre + Raise)
  - Example: $8M on $35M pre → $8M / $43M = **18.6%** (before option pool top-up)
- **Option pool top-up** usually comes **pre-money** (i.e., increases dilution to founders).

## Suggested Series A target structure (illustrative)
- New investors: ~18–22%
- Option pool: 10–15% post
- Founders + existing holders: remainder

## What to prepare for investors
- Current cap table (fully diluted)
- Scenario table: pre-money $30M/$35M/$40M and raise $6M/$8M/$10M
- Post-money ownership incl. option pool top-up
- Fully diluted share count + implied price per share
