## Creative Asset Requirements (Production-ready specs)
### Static image specs
- 1080×1080 (feed)
- 1080×1350 (recommended; higher engagement)
- 1080×1920 (stories/reels)

### Style
- Clean, clinical, non-sensational.
- Avoid “before/after” pain claims.
- Use neutral language: “confidential,” “respectful,” “private,” “assessment,” “plan.”

### Image concepts (briefs)
**Chiro:**
- Practitioner hands on upper back/neck (fully clothed, clinical)
- Patient doing mobility exercise
- Text overlay: “Chiropractic Care | Edmonton | Same-week appointments”

**Pelvic Women:**
- Calm clinic room, neutral lifestyle image (walking, lifting baby carrier without showing baby), or abstract “care/comfort” imagery
- Text overlay: “Pelvic Floor Therapy | Postpartum Support | Confidential Care”

**Pelvic Men:**
- Neutral men’s health imagery (walking, seated consultation), clinic room, or abstract graphic
- Text overlay: “Men’s Pelvic Health | Discreet Care | Edmonton”

## Launch Checklist (click-by-click)
1. Ads Manager → **Create**
2. Select **Objective** (Leads)
3. Name campaign exactly as above
4. Set daily budget
5. Create Ad Set → set location, age, gender, targeting
6. Choose placements (Advantage+)
7. Create ad:
   - Add creative (image/video)
   - Paste primary text/headline
   - Set CTA
   - Set destination URL (or select Instant Form)
8. Publish
9. Verify:
   - Form submits properly
   - Leads appear in Leads Center
   - Notifications enabled for new leads
